# Aula 11 - Cenário C: Figurinhas

## Como rodar

Entre na pasta:

```bash
cd flask/Aula11
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Rode o projeto:

```bash
python app.py
```

Abra no navegador:

```text
http://127.0.0.1:5000/figurinhas/
```

## Observação

Antes de entregar, troque `# Aluno: Seu Nome` pelo seu nome nos arquivos `.py`.
